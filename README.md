# htmldump

Dump scraped HTML from MongoDB to a bzip2-compressed tar file archive.

## Requirements

* Python 2.7
* Dependencies from `requirements.txt`

## Usage

    htmldump.py -H opented.org -u USERNAME -p PASSWORD OUTPUT [DOC-RE]

See `htmldump.py -h` for details.
